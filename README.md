# OkulPro — SearchableSelect + Gelişmiş Filtreler

YENİ:
1) SearchableSelect (Arama özellikli seçim kutusu)
   - 5'ten fazla seçenek varsa otomatik arama kutusu açılır
   - Yazarak filtreleme (örn: "Mat" → Matematik, Matematik 2 hızlı bulunur)
   - Klavye ile gezinme: ↑↓ Enter Esc
   - 120+ branş arasından öğretmen branşı seçimi artık şimşek hızında
   - Aktif seçimde altın sol kenar çizgisi + ✓ tik
   - Boş aramada "Sonuç yok" empty state
   - Mouse dışında tıklayınca otomatik kapanır

2) Yerel Kullanım:
   - Bilgiler: Öğretmen ekleme (branş dropdown), tüm form select'ler
   - Atamalar: Öğretmen, Sınıf, Ders, Blok, Derslik
   - Filtre çubuğu: Öğretmen filtresi (Gelişmiş içinde)

3) Gelişmiş Filtreler Pattern:
   - Üstte 2-3 ana filtre (Sınıf chip'leri — her zaman görünür)
   - "⚙️ Gelişmiş Filtreler ▾" butonu sağda
   - Tıklayınca açılır: Öğretmen filtresi vs. genişleyebilir
   - Aktif olunca altın highlight
   - "↺ Tüm Filtreleri Temizle" CTA

Vercel: Framework Preset = Other, build/install/output hepsi boş.
