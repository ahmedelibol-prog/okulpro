# OkulPro — Ders Programı Hazırlama Yazılımı

## Bu Klasörün İçeriği
Hepsi tek dosyada toplanmış statik dosyalar. Hiç build yok, klasör yok.

- `index.html` — Tüm uygulama (JS, CSS, her şey içinde)
- `vercel.json` — SPA routing
- `robots.txt`, `sitemap.xml` — SEO
- `Lisans_Yoneticisi.html` — Çağrı'ya özel lisans üretici (Masaüstüne taşı)

## Vercel Yükleme
1. Bu klasördeki **5 dosyayı** GitHub deposuna sürükle
2. Vercel otomatik deploy eder, hiçbir şey çalıştırmaz

## Vercel Ayarları
Settings → Build & Development Settings:
- Framework Preset: **Other**
- Build Command: **boş bırak**
- Output Directory: **boş bırak**
- Install Command: **boş bırak**
